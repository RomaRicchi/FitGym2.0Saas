2. Descripcion del Servicio

Flow Manager proporciona una plataforma de software como servicio (SaaS) para la gestion integral de negocios, que incluye:

  • Gestion de clientes y membresias
  • Control de accesos y check-ins
  • Gestion de pagos y suscripciones
  • Planificacion de rutinas y actividades
  • Reportes y estadisticas
  • Configuracion de personal y usuarios

El Servicio se proporciona "tal cual", sin garantias de ningun tipo, expresas o implicitas.

4. Cambios de Plan - Subir de Nivel

El Cliente puede solicitar subir de plan en cualquier momento.

El cambio solicitado NO es inmediato. El nuevo plan se activara automaticamente el primer dia del siguiente mes de facturacion.

Sin prorrateo: No se aplicaran ajustes prorrateados. El Cliente pagara la tarifa completa del nuevo plan desde el primer dia del siguiente mes.

Sin restricciones: No hay limite de veces que el Cliente puede subir de plan.

5. Cambios de Plan - Bajar de Nivel

Restriccion Importante

NO se permite bajar de plan si la cantidad actual de clientes activos supera el limite del plan inferior.

Ejemplo: Si el negocio tiene 300 clientes activos y desea cambiar al plan "Basic" que permite solo 50 clientes, el sistema no permitira el cambio directo.

Procedimiento para Bajar de Plan

Para poder cambiar a un plan inferior, el Cliente debe:

  1. Dar de baja clientes excedentes: Acceder a la seccion "Clientes" y cambiar el estado a "inactivo" de los clientes sobrantes hasta alcanzar el limite del nuevo plan.
  2. Verificar el limite: Asegurarse de que el numero de clientes activos sea igual o inferior al limite del plan deseado.
  3. Solicitar el cambio: Una vez cumplido el requisito anterior, el Cliente podra solicitar el cambio de plan desde la seccion "Configuracion SaaS > Cambiar Plan".

Politica de Datos Historicos

Los clientes dados de baja NO se eliminan de la base de datos. Se mantienen en el sistema con estado "inactivo" para preservar el historial de:

  • Pagos realizados
  • Asistencias registradas
  • Rutinas asignadas
  • Evolucion fisica
  • Historial de comunicaciones

El Cliente puede reactivar clientes en cualquier momento, siempre que no supere el limite de su plan actual. Si al reactivar se excede el limite, debera subir de plan o dar de baja otros clientes.

Prohibicion de Abuso

Esta expresamente prohibido: Contratar un plan superior para cargar masivamente datos de clientes y posteriormente bajar a un plan inferior intentando mantener dicha capacidad. Esta practica constituye abuso del servicio y puede resultar en:

  • Suspension del servicio
  • Cancelacion de la cuenta sin reembolso
  • Bloqueo permanente de la capacidad de bajar de plan

Vigencia de los Cambios

Los cambios de plan (tanto subidas como bajas) se aplican el primer dia del siguiente mes, independientemente de cuando se solicite el cambio. No se aplican prorrateos.

16. Renovacion Automatica de Planes

Todos los planes de suscripcion de Flow Manager se renuevan automaticamente al finalizar cada periodo de facturacion (mensual o anual), salvo que el Cliente cancele el servicio antes de la fecha de renovacion.

Proceso de Renovacion

  • Notificacion previa: Se enviara un email 7 dias antes de la fecha de renovacion con el importe a cobrar.
  • Cargo automatico: En la fecha de renovacion, se cobrara automaticamente el importe correspondiente al plan contratado mediante el metodo de pago registrado.
  • Comprobante de pago: El Cliente recibira un comprobante por email una vez procesado el pago exitosamente.
  • Continuidad del servicio: El servicio no se interrumpira mientras el pago se procese correctamente.

Pagos Rechazados

Si el pago automatico es rechazado por el procesador (tarjeta vencida, fondos insuficientes, etc.):

  • Primer intento fallido: Se notificara al Cliente por email y se realizara un segundo intento en 48 horas.
  • Segundo intento fallido: El servicio sera suspendido temporalmente. El Cliente tendra 7 dias para regularizar el pago.
  • Sin regularizacion despues de 7 dias: La cuenta sera cancelada y los datos seran archivados segun lo establecido en la Seccion 10.

Cambios de Plan y Renovacion

Si el Cliente solicita cambiar de plan antes de la fecha de renovacion:

  • Subir de plan: Se aplica el primer dia del siguiente mes.
  • Bajar de plan: Se aplica el primer dia del siguiente mes, siempre que cumpla con los requisitos de la Seccion 5.
  • Cancelar renovacion automatica: El Cliente puede optar por no renovar automaticamente desde "Configuracion SaaS > Cancelar Servicio".

17. Periodo de Prueba (Trial)

Flow Manager ofrece un periodo de prueba gratuito de 14 dias para que los nuevos clientes evaluen el servicio antes de comprometerse a un plan pago.

Caracteristicas del Periodo de Prueba

  • Duracion: 14 dias corridos desde la fecha de registro
  • Costo: 100% gratuito, sin compromiso
  • Funcionalidades completas: Acceso a todas las caracteristicas del plan seleccionado durante el trial
  • Sin tarjeta de credito: No es necesario ingresar datos de pago para iniciar el trial
  • Limite de clientes: Puedes agregar hasta 50 clientes durante el periodo de prueba
  • Soporte incluido: Acceso a soporte tecnico durante el trial

Datos del Periodo de Prueba

Durante el trial, puedes:

  • Crear clientes, asignar rutinas y registrar asistencias
  • Configurar tu negocio, salas y personal
  • Probar todas las funcionalidades del sistema
  • Generar reportes y estadisticas

Importante sobre los datos: Todos los datos que cargues durante el periodo de prueba se mantendran al finalizar el trial. No necesitas volver a cargar nada.

Fin del Periodo de Prueba

Al finalizar los 14 dias de prueba:

  • Sin accion del Cliente: Si no seleccionaste un plan ni agregaste datos de pago, la cuenta pasara a estado "Suspendido". Los datos se mantendran por 30 dias antes de ser eliminados.
  • Con plan seleccionado: Si ya elegiste un plan e ingresaste datos de pago, la suscripcion se activara automaticamente y se realizara el primer cobro.
  • Suscripcion anticipada: Puedes suscribirte a un plan pago en cualquier momento durante el trial. El periodo de prueba finalizara y comenzara la facturacion del plan.

Un trial por cliente: Cada Cliente (negocio) puede disfrutar de UN unico periodo de prueba gratuito. No es posible reiniciar el trial despues de finalizado.
